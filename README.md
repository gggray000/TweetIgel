# TweetIgel

<img src="tweetigel-frontend/public/tweetigel_logo.png" alt="Tweetigel Logo" width="100"/>

## Introduction
This is a microblog project for a university course "Web Technology Project". <br>
Technology stack: <br>
- Backend:Java Spring Boot
- Frontend: React.js
- Database: MariaDB
- Deployment: Docker Compose<br>

## Usage
- run <code>docker-compose up -d --build</code>, and access the frontend via <code>localhost:80</code>.
- Test data can be populated to the application by running <code>testData.sql</code>.

## Features
1. Account register and login.
2. Send posts.
3. Search other users.
4. Follow and unfollow other users.
5. Like and unlike posts.
6. Manage one's own user profile.
7. Display user profile of another user, including their posts.
8. Present the timelines of posts as a paginated view.
9. Delete one's own posts.
10. Comment on posts.
11. Tag posts using a hashtag mechanism.
12. Retrieve posts based on hashtags.
